**Что сделано:**

**Технологии:** Python, SQL, ClickHouse, Airflow

## :whale: DevOps Lab

**Описание:** Учебная лаборатория по DevOps — Docker, CI/CD, мониторинг.

**Что сделано:**

- Изучил Docker: образы, контейнеры, тома, Dockerfile
- Собрал Flask-приложение в контейнере
- Настроил CI/CD на GitHub Actions с пушем в Docker Hub
- Развернул стек мониторинга: Node Exporter → Prometheus → Grafana
- Дашборд с метриками CPU, памяти, диска

**Технологии:** Docker, GitHub Actions, Prometheus, Grafana, Python/Flask

**Репозиторий:** [devops-lab-latanov](https://github.com/Tsukuyomiwalk/devops-lab-latanov)
